# 🏠 California Housing Price Prediction

## Overview

This project focuses on predicting housing prices in California using machine learning algorithms.

The goal is to build a regression model that can estimate house prices based on geographical and demographic features.

---

## Dataset

The dataset contains information about California districts:

Features:

- Median income
- House age
- Average rooms
- Population
- Latitude
- Longitude

Target:

- Median house value

---

## Technologies

Python

Libraries:

- Pandas
- NumPy
- Matplotlib
- Scikit-learn

The project includes:

✅ Data exploration

✅ Model training  
✅ Model evaluation  

---

## Machine Learning Models

Models tested:

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

---

## Evaluation

Metrics:

- MAE
- RMSE
- R² Score

---

## Results

The final model achieved good prediction performance and demonstrated how machine learning can be applied to real estate price prediction.
