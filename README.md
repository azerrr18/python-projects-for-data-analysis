# 📂 Projects

## 🏠 California House Price Prediction

### Overview

A regression machine learning project that predicts California house prices based on demographic and geographical features.

The goal of this project is to build a model that can estimate housing prices using historical data.

---

### Dataset

The dataset contains information about California districts:

Features:

- Median Income
- Housing Median Age
- Average Rooms
- Average Bedrooms
- Population
- Latitude
- Longitude

Target:

- Median House Value

---

### Workflow

The project includes:

✅ Data exploration  
✅ Data visualization  
✅ Data preprocessing  
✅ Feature analysis  
✅ Model training  
✅ Model evaluation  

---

### Machine Learning Models

Models tested:

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

---

### Evaluation Metrics

Models were evaluated using:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score
